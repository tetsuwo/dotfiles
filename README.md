dotfiles
========


### 1. Download

```
~$ git clone git@github.com:tetsuwo/dotfiles.git ~/dotfiles
  or
~$ git clone https://github.com/tetsuwo/dotfiles.git ~/dotfiles
```


### 2. Install

```
~$ cd ~
~$ sh dotfiles/scripts/install.sh
```


### 3. Update

```
~$ cd ~
~$ sh dotfiles/scripts/update.sh
```
